pytest-cocotb
=============

A `pytest <https://docs.pytest.org>`_ plugin that integrates
`cocotb <https://www.cocotb.org>`_ HDL simulation into your normal pytest
workflow.  Compile once per session, then run individual cocotb testbenches
as regular pytest test cases with full parametrisation, fixture, and
reporting support.
